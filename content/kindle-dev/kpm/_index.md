# KPM
KPM is the Kindle Package Manager, it was created by [Hackerdude](https://hackerdude.tech), as of `hdnext` it is the recommended distribution method for Kindle homebrew.

## Understanding KPM
KPM is a package manager, it is a tool which indexes Kindle homebrew and allows you to download and install it on-device.  
Each developer is expected to submit to the official [KindleModding KPM repository](https://github.com/KindleModding/repo/pulls) or host their own repository (repositories can be hosted statically on GitHub pages)
